Heavy Pinky
A tiny browser extension that fixes accidental second-letter capitalization (like "THe" → "The") while you type.

Why
Fast typers sometimes hold Shift a fraction too long and end up with words like "THe" or "THis". Heavy Pinky quietly corrects these mistakes.

Features
• Automatic correction at sentence start
• Optional aggressive mode
• Works in text inputs, textareas, and editable fields
• No tracking, no analytics
• Runs entirely locally in your browser

Install
Clone the repo and load it as an unpacked extension.

Chrome:
1. Open chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select the Heavy Pinky folder

Privacy
Heavy Pinky does not collect, transmit, or store any personal data.
